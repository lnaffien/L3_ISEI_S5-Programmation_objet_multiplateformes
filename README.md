# L3_ISEI_S5-Programmation_objet_multiplateformes

## Projet obligatoire
Créer un programme permettant de gérer des matrices sous 3 formes différentes : 
- Matrices denses
- Matrices creuses
- Matrices linéaires

Toute matrice d'un type spécifique pourra être transformée en un autre type.
